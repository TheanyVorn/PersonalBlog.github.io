Text to Speech Convertor

Text to speech (TTS) is a technology that converts written text into spoken words. It is used in various applications, including screen readers, virtual assistants, and language learning tools.

In this project, we will create a simple TTS application using Python and the gTTS library. The gTTS library allows us to convert text into speech using Google's Text-to-Speech API.
